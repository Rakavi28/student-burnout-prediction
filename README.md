# Student Burnout Prediction

Behavioural analytics project to predict student burnout risk using machine learning.

## 📌 Problem Statement
Student burnout is a growing concern in academic environments.  
This project aims to identify students at risk using behavioural and lifestyle indicators.

## 📊 Dataset
A synthetic dataset was generated to simulate student behaviour patterns.

Features include:
- Study hours
- Sleep hours
- Stress level
- Physical activity
- Social interaction
- Academic performance

Target variable:
- Burnout Level (Low, Medium, High)

## ⚙️ Methodology
1. Data generation using Python
2. Data preprocessing
3. Exploratory Data Analysis (EDA)
4. Model training using Logistic Regression
5. Model evaluation using accuracy

## 📈 Results
The model successfully predicts burnout levels based on behavioural factors, showing that:

- High stress + low sleep strongly correlate with burnout
- Physical activity reduces burnout risk

## 🧠 Insights
Behavioural analytics can help institutions identify at-risk students early and implement preventive strategies.
